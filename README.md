👙 華歌爾 中壢課智能排班系統 (Wacoal Smart Schedule System)

專為華歌爾桃園販賣部（中壢課）量身打造的現代化「卡片式沙盤推演」與「自動排班」ERP 系統。解決傳統 Excel 排班耗時、易撞期、且難以檢核「雙週變形工時」法規的痛點。

✨ 核心特色 (Key Features)

1. 🗂️ 企業級架構與資產管理

課區與組別層級：支援多課區建檔，完整劃分「貞文組」、「秋玉組」編制。

人員 / 店家 CRUD：已預載中壢課 32 名員工與 17 間店舖完整資料。支援設定職務（組長/員工/機動）、狀態（正常/病假/育嬰假）、店鋪通路、營業時間與店公休日。

2. 🃏 沙盤式卡片分派 (Drag & Drop)

無限人員庫：左側待分派區可無限次拖拉同一位員工，完美支援「一人支援多店」的機動零售生態。

動態主副櫃判斷：拖入店家陣地的第一人自動標記為【主負責】，其餘為【支援】。

3. 🧠 智能自動排班引擎 (V10 核心)

法規對齊：內建「全局雙週基準日」設定（預設 2025-07-07 星期一），排班引擎嚴格依據 14 天週期鋪排公休，避免跨月違法。

防分身撞期：針對跨店支援人員，系統自動檢查當日是否已在 A 店排班，避免在 B 店產生「幽靈班表」。

A/P/O 互補配置：依據各店「派員數」設定，自動運算早班 (A)、晚班 (P) 與全班 (O) 的合理搭配。組長職務則自動略過不排班。

4. 📊 四大排班視角與匯出

提供 全課角度、組長角度（置頂該組組長）、店家角度、人員角度 四種切換視角，雙週邊界皆有加粗線條提示。

支援點擊表格直接手動微調班別（A/P/O/X/特休/代）。

一鍵匯出符合企業傳統格式的 Excel 報表。

🛠️ 技術架構 (Tech Stack)

本系統目前採用 「零安裝、純前端單一檔案」 架構，極度輕量且易於部署：

核心語言：HTML5, Vanilla JavaScript (無須依賴 React/Vue 等重型框架)

UI 樣式：Tailwind CSS (透過 CDN 載入)

圖示庫：Phosphor Icons

Excel 處理：SheetJS

🚀 快速開始 (Quick Start)

階段一：本機 / GitHub Pages 測試 (目前階段)

無需安裝任何 Node.js 環境或資料庫。

直接使用瀏覽器（推薦 Chrome / Edge / Safari）開啟專案根目錄下的 index.html 檔案。

或將程式碼推播至 GitHub，並開啟 GitHub Pages 功能，即可獲得公開測試網址。

⚠️ 注意：目前處於 UI/UX 測試階段（Beta），重新整理網頁後資料會恢復為預設的預載狀態。
